# GfxHealthCheck
A tool to collect and analyse different gpu related info and help fix gfx related problems and setup system properly

# requirements
- python 3.5
- cmake 3.7
- gcc 4.8.1

# build
```bash
mkdir build && cd build && cmake .. && make -j8
```

# usage
after build 
```bash
python3 gfxhealthcheck.py
```
